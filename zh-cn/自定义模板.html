自定义模板

可以使用任意html文件替换内部模板。首先修改配置文件：

module.exports = {
  rootDir: './docs',
  outDir: './docsite',
  plugins: {},
  template: 'new_template.html' // specify template file
}

其次请申明相关注入点。

注入点

注入点是具备固定id的DOM元素，Docp会将解析结果插入到这些DOM元素中。目前开放的注入点包括：顶部导航、菜单、内容区域。

导航注入点：#docp-header

导航信息来自docp.config.js的header属性，请参考生成顶部导航。

目录注入点：#docp-menu

菜单信息来自SUMMARY.md，请参考生成目录。

内容区域注入点：#docp-content

该注入点必须提供。
